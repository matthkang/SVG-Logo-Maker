# SVG Logo Maker

## Description

Create a logo.svg with custom text and color based on user input

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Tests](#tests)
- [Questions](#questions)

## Installation

To install dependencies:
```npm i```

## Usage

To run application:
```node index.js```
